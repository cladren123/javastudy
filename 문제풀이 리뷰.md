트리문제를  풀 때는 트리를 탈 이유가 없을 때 중단하는 조건을 추가해야 한다. 아니면 의미없는 작업을 계속 반복해 시간초과가 날 수 있다.

## 2차원 배열





90도 회전하기

```java
int n = 3;
int[][] arr = new int[n][n];
int count = 1;
for(int i = 0; i < n; i++)
    for(int j = 0; j < n; j++)
        arr[i][j] = count++;

int[][] rotate = new int[n][n];
for(int i = 0; i < n; i++)
    for(int j = 0; j < n; j++)
        rotate[i][j] = arr[n-1-j][i];
        
```
