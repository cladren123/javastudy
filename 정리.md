## 파일 명명 규칙

1. 패키지명은 영문소문자를 사용
2. 2개 이상 단어 조합시 Camel 규칙 적용 
3. _ 사용 금지, 의미 있는 명사로 사용 





## 백준에러

* 컴파일 에러

클래스 이름을 Main으로 바꿔줘야 한다.

public class Main {



}





* 런타임 에러 main class Main



main.java는 다음과 같이 작성되어야 한다.

```java
class Main {
    public static void main(String[] args) {
        
    }
}
```



해결방안

1. package를 지우면 해결

2. void 사용 (int main을 사용하면 안됨)
3. static으로 선언해야 한다.
4. main으로 사용해야 한다.
